I have also used and rougly adapt a piece of code for parsing a youtube playlist url, found here

Execute this script in the directory in which songs have to be downloaded (e.g: Create a 'Myplaylist' directory and from the terminal):

cd /PathToMyPlaylist/Myplaylist  <= to be in the right directory
python /PathToTheScriptFolder/playlist_downloader/dl_ytpl_multi.py http://youtubePlaylistAdress.com/343424324...

It creates a playlist.txt file, do not delete or edit this file if you don't want to download the same video each time you run the script (it records all downloaded viedos) from a given playlist.
